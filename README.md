# 📸 Photographer - VulnHub Walkthrough

> Projeto acadêmico de Cybersecurity focado na análise e exploração de vulnerabilidades da máquina virtual **Photographer**, disponibilizada pela plataforma VulnHub para treinamento em Pentest e Segurança Ofensiva.

---

## 📖 Sobre o Projeto

Este projeto documenta a realização de um teste de invasão completo em ambiente controlado utilizando a máquina vulnerável Photographer, da plataforma VulnHub.

O laboratório simula um cenário realista de avaliação de segurança, exigindo a aplicação de técnicas de reconhecimento, enumeração, exploração e escalonamento de privilégios para obtenção de acesso administrativo ao sistema.

---

## 🎯 Objetivos

* Desenvolver habilidades práticas em Pentest
* Aplicar metodologias de Ethical Hacking
* Exercitar técnicas de enumeração de serviços
* Realizar análise de vulnerabilidades
* Compreender processos de Privilege Escalation
* Produzir documentação técnica profissional

---

## 🛠️ Tecnologias e Ferramentas

| Categoria           | Ferramentas         |
| ------------------- | ------------------- |
| Sistema Operacional | Linux               |
| Reconhecimento      | Nmap                |
| Enumeração Web      | Gobuster            |
| Comunicação         | Netcat              |
| Ambiente            | VulnHub             |
| Virtualização       | VirtualBox / VMware |
| Documentação        | Markdown e PDF      |

## 🔍 Escopo do Laboratório

* Reconhecimento da superfície de ataque

* Coleta de evidências
